# Website_Assignment
#### Gym Website UI

This project is a responsive Gym Website User Interface built using HTML, Bootstrap 5, CSS, and JavaScript. The website includes essential sections commonly found in fitness websites such as a homepage with a carousel, services cards, an about section, registration form, and login validation.

The UI is designed to provide a clean and modern layout for a fitness or gym platform where users can explore services, learn about the gym, and register or log in.


## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Steps to Run the Project Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/gym-website-ui.git
   ```

2. **Navigate to the project folder**

   ```bash
   cd gym-website-ui
   ```

3. **Open the project in a code editor**

   You can open the folder in editors like **VS Code**.

4. **Run the project**

   Open the `home.html` (or `index.html`) file in your browser.

   * Double-click the file
     **or**
   * Right-click → **Open with Browser**

5. **Optional (Recommended)**
   If you are using **VS Code**, install the **Live Server extension** and run the project by clicking **“Go Live”** for a better development experience.

The website will open in your browser and you can navigate through the pages such as Home, About, Register, and Login.
